这篇文档介绍如何在Android和Ubuntu下访问GPIO。

准备工作

固件版本必须满足以下条件：

  • Android M >= V170603
  • Android N >= V170421
  • Ubuntu Server >= V180712
  • Ubuntu Mate >= V180531

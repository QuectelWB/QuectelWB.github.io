下载NPU相关工具包SDK

请访问这里来获取SDK下载链接。

下载NPU相关SDK到某个目录，如：~/npu

目录说明：

  • docs：模型转换说明文档
  • acuity-toolkit：模型转换相关工具
  • linux_sdk：Linux SDK
  • android_sdk：Android SDK

环境搭建

要使用模型转换工具必须要先安装TensorFlow等工具。

主机环境要求：

  • Ubuntu 16.04 X64
  • Ubuntu 18.04 X64

注意：默认安装的TensorFlow为CPU版本，如果你的电脑有英伟达GPU，你也可以选择安装GPU版本来加速模型转换速度。

验证TensorFlow安装：

如果看到Hello World打印说明TensorFlow安装成功。
